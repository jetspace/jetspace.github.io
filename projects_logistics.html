Logistics

GTK-based Pacman frontend



About

Logistics is a small GTK-based frontend for Pacman. Pacman is the default package manager of ArchLinux, so it handels installed packages and updates.
Logistics aims to provide users a simple way to manage packages via GUI and without using the terminal
