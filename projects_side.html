SiDE

Simple Desktop Environment



About

SiDE is an all new desktop environment. It aims to provide a known interface by maintaining a classical desktop interface. This means that you have one panel, a wallpaper and an application menu. Thats all, no fancy tiles or something like this.

But this does not mean, that SiDE is old-fashioned. SiDE is build on modern technologies like GTK3. A lot of the librarys used are developed by us.
SiDE comes ready to use with all important applications like text-editors.
