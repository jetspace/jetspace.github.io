Welcome to the JetSpace Project
So what is JetSpace?

We're trying to create an ArchLinux based Linux OS. But we are far away from that goal right now... At this time we are writing the basics of the tools we may need... For Example the installer (which is of course the most important) is not nearly stable. But we will keep on working. Anyways if you want to help JetSpace, you can! JetSpace is compleatly Open Source Software (OSS) and we would welcome everyone who wants to develope, test, design, write documentations and so on...
