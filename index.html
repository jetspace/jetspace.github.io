Linux

JetSpace is built on Linux! Linux is an open source project which was founded by Linus Torvalds. Today it is one of the most important operating systems in the world! Even large companies are using Linux on their infrastructure!
The open source developement has a lot of advantages for the users. For example Linux is very safe and there are almost no viruses for Linux based operating systems. This means that you do not have to install an anti virus tool.

This will save you money but also resources on your computer, because there is one application less running in the background!
And resources is the next advantage: Linux does not only run on new modern systems, it has quite low hardware requirements. JetSpace will follow the model of ArchLinux and we will support all computers which support x86 or x68_64!

Userfriendly

While ArchLinux™ is super fast and up to date, it can be quite hard for new users to understand how to use it! So we thought it would be good if there would be a way to get all the advantages of ArchLinux but also provide a simple user interface...

And then we created JetSpace! It should keep the advantages of ArchLinux but should also provide sytem level tools like an installer which should enable new users to use these advantages!

SiDE

JetSpace comes with a all new desktop enviroment: SiDE. SiDE is designed to work on low powered hardware but should still have a nice UI. So it basicly follows the same goals as JetSpace. SiDE is built with GTK3 which is a stable framework for user interfaces.

As the project continues we also try to implement as much XDG standards as possible to enable you to have a great user experience even across multiple enviroments!
