SiDE is a Simple Desktop Enviroment (SImple Desktop Enviroment). It is very simple but it has a modern layout and design. It has a few Programms installed :
- SiDE Edit -> A simple Editor
- SiDE View -> A Picture viewer
- SiDE Files -> A File Manager
